## Базовый датчик случайных чисел

### 1. Цель работы

Реализовать базовый датчик случайных чисел на основе линейного конгруэнтного метода. Вычислить выборочное среднее и дисперсию для сгенерированной последовательности, а также для последовательности, полученной с помощью встроенного генератора языка программирования. Сравнить полученные статистические характеристики с теоретическими значениями для равномерного распределения на интервале $[0, 1]$. Сделать вывод о качестве работы реализованного и встроенного генераторов.

---

### 2. Описание генераторов

#### 2.1. Реализованный генератор

Линейный конгруэнтный генератор – один из простейших алгоритмов генерации псевдослучайных чисел. Он описывается рекуррентным соотношением:

$$X_{n+1} = (a \cdot X_n + c) \mod m$$

В данной работе использованы параметры:
- $a = 1664525$ (множитель)
- $c = 1$ (приращение)
- $m = 2^{32}$ (модуль)
- начальное состояние (seed) $X_0 = 22$

Для получения случайного числа в интервале $[0, 1)$ значение $X_n$ делится на $m$:

$$U_n = \frac{X_n}{m}$$

#### 2.2. Встроенный генератор Python

Встроенный генератор `random.random()` реализован на основе вихря Мерсенна с периодом $2^{19937}-1$ и обеспечивает равномерное распределение на интервале $[0, 1)$. Этот метод считается стандартом для многих приложений.

---

### 3. Вычисляемые характеристики

Для выборки $\{U_1, U_2, \dots, U_N\}$ размера $N$ вычисляются:

- **Выборочное среднее**:
  $$\overline{U} = \frac{1}{N}\sum_{i=1}^{N} U_i$$

- **Выборочная дисперсия**:
  $$S^2 = \frac{1}{N-1}\sum_{i=1}^{N} (U_i - \overline{U})^2$$

Теоретические значения для равномерного распределения $U \sim (0,1)$:
- Математическое ожидание: $\mu = 0.5$
- Дисперсия: $\sigma^2 = \frac{1}{12} \approx 0.0833333$
